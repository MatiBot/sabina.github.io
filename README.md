# Sabina Rovansek - UX Designer Portfolio

A professional yet playful portfolio website for UX designer Sabina Rovansek.

## Features

- Clean, modern design with playful touches
- Fully responsive layout
- Smooth scrolling navigation
- Animated hero section decorations
- Professional portfolio showcase
- Contact section with social links

## GitHub Pages Setup

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" in the sidebar
4. Under "Source", select your branch (usually `main` or `master`)
5. Select the root folder (`/`)
6. Click "Save"
7. Your site will be available at `https://[username].github.io/[repository-name]`

## Customization

- Update email and social links in `index.html`
- Modify the portfolio projects in the work section
- Adjust colors in the CSS variables at the top of `styles.css`
- Replace placeholder work images with actual project screenshots

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (for smooth scrolling and navbar effects)
- Google Fonts (Inter & Playfair Display)
